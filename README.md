# 🌳 Binary Tree Simulator

> **A visual, interactive Binary Tree playground built for people who actually want to *understand* DSA. Not just memorize it.**

🔗 **Live Demo:** [keerthisuryateja.github.io/Binary-Tree-Simulator](https://keerthisuryateja.github.io/Binary-Tree-Simulator/)

---

## 🤌 What even is this?

You know that feeling when your professor draws a binary tree on the board and you're just nodding like you get it, but inside... nothing? Yeah. This fixes that.

**Binary Tree Simulator** is a browser based, zero setup visual tool that lets you:

- 🌱 **Insert, Delete, and Find** nodes in a BST. And actually *see* what's happening
- 🔄 **Run Inorder, Preorder, and Postorder traversals** with step by step animation
- 📜 **Algorithm trace panel** that shows the pseudocode live as it executes
- 🌲 **Load preset trees** so you don't have to type values every time
- ⚡ **Control animation speed**. Slow it down when you're learning, speed it up when you're flexing
- 🎮 Runs fully in your browser. Just hit the live link, no local setup needed

## 🚀 Getting Started

**Option 1. Just use it (recommended tbh)**

Go to 👉 [keerthisuryateja.github.io/Binary-Tree-Simulator](https://keerthisuryateja.github.io/Binary-Tree-Simulator/)

That's it. Seriously.

---

**Option 2. Run it locally**

```bash
git clone https://github.com/keerthisuryateja/Binary-Tree-Simulator.git
cd Binary-Tree-Simulator
npm install
npm run dev
```

Then open localhost in your browser. Needs Node.js installed.

## 🎯 Features

| Feature | What it does |
|---|---|
| 🌱 Insert Node | Add values and watch the tree build itself |
| 🗑️ Delete Node | Remove nodes with proper BST deletion logic |
| 🔍 Find Node | Search and highlight the path taken |
| 🔄 Traversals | Inorder / Preorder / Postorder with live animation |
| 📜 Algorithm Trace | Pseudocode panel highlights current step |
| 🌲 Preset Trees | Load example trees instantly |
| ⚡ Speed Control | Adjust animation from Fast to Slow |
| 🎨 Forest theme | Because plain white backgrounds are boring |
